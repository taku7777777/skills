# 企業向け文書共有サービス

## 目的

企業テナント内で機密文書をアップロードし、期限付きリンクで社外へ共有する。初年度1万テナント、平常100req/s、月末ピーク10倍を想定する。

## 要求

- 文書は最大2GB。アップロード完了後5分以内にウイルススキャンする。
- ダウンロードは99.95%、p95 2秒以内。アップロード処理は一時遅延を許容する。
- テナント間漏洩と期限後ダウンロードはゼロ許容。
- 共有リンクは受信者のメールアドレスに限定できる。
- 顧客指定で7年間保存する文書と、30日で削除する文書がある。
- EU顧客のデータはEUリージョンへ保存する。
- 初年度のクラウド費用上限は月800万円。

## 外部依存

- オブジェクトストレージ、CDN、メール配信、マルウェアスキャナ。
- 認証は企業IdPのOIDC。社外受信者はワンタイムコード。
