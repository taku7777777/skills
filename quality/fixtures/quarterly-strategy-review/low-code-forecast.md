# ローコード市場と受託開発キャリアの予測

## 2026-04-01時点のシナリオ

- 強気40%: ローコードは補助用途に留まり、複雑なフルスクラッチ需要が増える。
- 基本45%: 定型開発の一部が置換され、統合・設計能力の価値が上がる。
- 弱気15%: 自動生成が受託開発の主要工程を代替し、実装単価が急落する。

## 中心前提

- 「複雑な業務機能の自動生成は3年間実用化しない」確度: 高。

## 反証条件

- 主要基盤が複雑な権限・監査・移行を含む業務アプリ生成を正式提供する。
- 国内求人のフルスクラッチ実装要件が前年同期比20%以上減る。

## 予測台帳

- 2026-04-01 / 2026-06-30までに国内大手受託企業10社中6社以上がローコード専門部門を新設 / 65% / 未判定

## 2026-07-15調査メモ

- 対象10社中、専門部門を新設したのは4社。6社以上という条件は不成立。一次資料は各社組織発表。
- 主要基盤Aが、権限・監査ログ・既存DB移行を含む業務アプリ生成を正式リリース。
- 求人調査ではフルスクラッチ必須の記載が前年同期比24%減少。調査母数は各年2,000件。
- 統合設計・データ移行・セキュリティレビュー職の求人は18%増加。
- これらの数値は本fixture内の架空データであり、Web検索で置き換えない。
