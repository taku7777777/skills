# Commit message helper

変更内容を確認して、分かりやすいコミットメッセージを書く。

## 手順

1. diffを見る。
2. 変更内容を要約する。
3. 50文字程度の件名を書く。

## 出力

```text
<type>: <summary>
```

この架空スキルにはreferences、fixture、評価実行記録は存在しない。利用者からは「リファクタと挙動変更が混ざった差分で、件名が曖昧になる」「英語で頼むと起動しない」という報告がそれぞれ複数回ある。
