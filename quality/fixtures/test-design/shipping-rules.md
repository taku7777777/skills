# 配送料判定ルールとテスト条件

## 入力因子

- 会員区分: `regular` / `premium`
- 商品小計: 0円以上の整数
- 配送地域: `domestic` / `remote-island` / `international`
- 危険物: あり / なし
- 配送日: 平日 / 週末

## ルール

1. 国内通常配送は500円。
2. 国内で小計5,000円以上なら送料無料。
3. premium会員は国内の送料無料閾値が3,000円。
4. 離島は国内送料に1,200円を加算し、送料無料にならない。
5. 国際配送は2,500円で、危険物は配送不可。
6. 国内の危険物は800円を加算する。
7. 週末配送は配送可能な注文に300円を加算する。

## test-analysisから引き継いだ条件

| ID | 条件 | 優先度 | オラクル |
|---|---|---|---|
| TC-01 | regular国内の送料無料境界 | 高 | 4,999円=500円、5,000円=0円 |
| TC-02 | premium国内の送料無料境界 | 高 | 2,999円=500円、3,000円=0円 |
| TC-03 | 離島では閾値以上でも無料にならない | 高 | 基本送料+1,200円 |
| TC-04 | 国際×危険物を拒否する | 高 | 配送不可エラー |
| TC-05 | 週末加算と他ルールの組み合わせ | 中 | 配送可能なら最終送料+300円 |

制約: 25ケース以内に抑える。小計0円は有効入力。負数は入力検証層で拒否される。
